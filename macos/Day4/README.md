# Day 4 - Flutter Layout Widgets

## Overview

On Day 4, I learned Flutter Layout Widgets and practiced building responsive user interfaces. The focus was on understanding how widgets are arranged on the screen and how Flutter handles layout constraints.

* Build responsive layouts

* Create complex user interfaces
