# Flutter Demo App

This is a Flutter demo application developed during internship training.
It demonstrates basic Flutter setup, project structure, and UI components.

## Tech Stack
- Flutter
- Dart
- Android Studio / VS Code

## How to Run

1. Clone the repository  
   git clone https://github.com/SnehaTumaskar-16/Development.git

2. Navigate to project folder  
   cd demo

3. Install dependencies  
   flutter pub get

4. Run the application  
   flutter run
