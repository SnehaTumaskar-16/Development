# Task 1 - Android Studio & Flutter Installation

## Objective
Install Android Studio and Flutter SDK and set up the development environment.

## Tools Installed
- Android Studio
- Flutter SDK
- Dart Plugin
- Android Emulator

## Steps Followed

### 1. Android Studio Installation
- Downloaded Android Studio from the official website
- Installed with default settings
- Set up Android SDK and AVD (Emulator)

### 2. Flutter Installation
- Downloaded Flutter SDK from flutter.dev
- Extracted and set the Flutter path in Environment Variables
- Ran `flutter doctor` to verify installation

## Verification
- `flutter doctor` command ran successfully with no errors
- Created and ran a sample Flutter app on the emulator
